# Atividade - Vetores em JavaScript

## Descrição

Esta atividade tem como objetivo praticar o uso de vetores, estruturas de repetição e estruturas de decisão em JavaScript.

## Exercício 1

Programa que lê a idade de 4 pessoas, armazena os valores em um vetor e exibe a maior idade digitada e sua posição.

## Exercício 2

Programa que lê o nome, sexo e salário de 5 funcionários, armazenando as informações em vetores. Ao final, exibe apenas as funcionárias que possuem salário superior a R$ 5.000,00.

## Exercício 3

Programa que lê nomes e alturas de várias pessoas, permitindo que o usuário escolha quando encerrar os cadastros. Ao final, exibe a maior e a menor altura registradas e os respectivos nomes.
